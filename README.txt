Enables LiveTex widget on all pages.

== Description ==
This plugin adds the required javascript for LiveTex widget.

For more information visit: [LiveTex](https://livetex.ru/)

== Installation ==
1. Upload `simple-livetex` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the LiveTex ID from LiveTex site to the settings (Admin > Settings > Simple LiveTex).
